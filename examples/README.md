# Examples

### Requirements:

- Get a Kiln API key
- Get a Fireblocks API key & secret key

### Install dependencies:

```bash
bun install
```

### Setup your environment:
Fill your `.env` file with your Kiln and Fireblocks API keys.
```
cp .env.example .env
```

### Run an example:

```bash
bun run near.ts
```
